Contrôle du Flux & Boucles

« Guide ton aventure JavaScript : prends le bon chemin, déjoue les pièges, et fais tourner la boucle du destin ! »
L’art du contrôle des flux, t’offre la sagesse des choix dans le code : avancer, esquiver, répéter… que la logique soit ton épée !
Quête 1 : Contrôle des flux
  • Ton code avance étape par étape, mais tu peux changer la route avec des conditions et des boucles.

  • Utilise if, else, switch pour faire des choix.
Instructions conditionnelles
  • if : teste une épreuve (une condition)

  • else : suit la voie du destin si le test est raté

  • else if : propose des chemins alternatifs
let rubis = 30;

if (rubis > 50) {
  console.log("Assez pour acheter le bouclier d'or !");

} else if (rubis > 20) {
  console.log("Tu peux prendre une potion.");

} else {
  console.log("Va gagner plus de rubis !");
}
Conseil : Les accolades { ... } sont obligatoires pour ajouter plusieurs instructions à la condition !
Conditions avancées
  • Combine des tests avec && (ET), || (OU)
  • Utilise switch pour gérer plusieurs voies prédéfinies
let météo = "orage";

switch (météo) {

  case "soleil":
    console.log("Sors ton épée, libre !");
    break;

  case "pluie":
    console.log("Attention, sol glissant.");
    break;

  case "orage":
    console.log("Range tout métal, prudent héros.");
    break;

  default:
    console.log("Décision au feeling !");
}
Nota bene : switch est idéal quand tu attends plusieurs choix possibles connus (comme la météo, un rang, un type d’arme…)

break est essentiel pour éviter que le code continue dans les autres cas.

default gère le cas où aucune condition n’est remplie.
Approfondir les conditions
  • Teste plusieurs conditions imbriquées
  • Utilise ! pour l’inverse
  • Attention aux parenthèses pour clarifier la logique !
let héros = "Link";
let vie = 11;

if (héros === "Link" && vie > 10) {
  console.log("L’aventure continue !");
}

if (!(vie < 5)) {
  console.log("Encore assez de cœur !");
}
Astuce : Avec !, tu inverses la condition (« pas… »).

Avec plusieurs if imbriqués, tu gères les situations les plus complexes de la quête.
Les boucles (la vraie aventure)
  • for : pour répéter une action (parcourir un inventaire…)
  • while : tant qu’une condition est vraie, la boucle tourne
  • do...while : boucle exécutée au moins une fois (piège, mini-jeu !)
let inventaire = ["épée", "potion", "rubis"];

for(let i = 0; i < inventaire.length; i++){
  console.log("Objet trouvé : " + inventaire[i]);
}


let cœurs = 3;

while(cœurs > 0){
  console.log("♥ : " + cœurs);
  cœurs--;
}
Conseil : Attention aux « boucles infinies » : vérifie que ta condition change bien à chaque tour !

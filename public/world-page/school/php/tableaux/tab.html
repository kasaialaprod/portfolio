Tableaux et structures de données

Tableaux Fonctions

Introduction aux tableaux

Les tableaux (arrays) sont des structures de données qui permettent de stocker plusieurs valeurs dans une seule variable.

Structure d'un tableau

[0]
"Pomme"
[1]
"Banane"
[2]
"Orange"
[3]
"Kiwi"

Tableau indexé - Accès par position numérique

Tableaux indexés

Les tableaux indexés utilisent des indices numériques pour accéder aux éléments.

Création d'un tableau indexé

<?php
// Méthode 1: avec array()
$fruits = array("Pomme", "Banane", "Orange");

// Méthode 2: avec [] (PHP 5.4+)
$couleurs = ["Rouge", "Vert", "Bleu"];

// Méthode 3: assignation manuelle
$jours[0] = "Lundi";
$jours[1] = "Mardi";
$jours[2] = "Mercredi";
?>

Accès aux éléments

<?php
$fruits = ["Pomme", "Banane", "Orange"];

echo $fruits[0]; // Affiche: Pomme
echo $fruits[1]; // Affiche: Banane
echo $fruits[2]; // Affiche: Orange

// Modification d'un élément
$fruits[1] = "Kiwi";
echo $fruits[1]; // Affiche: Kiwi
?>

Tableaux associatifs

Les tableaux associatifs utilisent des clés nommées au lieu d'indices numériques.

<?php
// Création d'un tableau associatif
$personne = [
    "nom" => "Dupont",
    "prenom" => "Jean",
    "age" => 25,
    "ville" => "Paris"
];

// Accès aux éléments
echo $personne["nom"]; // Affiche: Dupont
echo $personne["age"]; // Affiche: 25

// Modification
$personne["age"] = 26;
?>

Tableau associatif

["nom"]
"Dupont"
["prenom"]
"Jean"
["age"]
25
["ville"]
"Paris"

Accès par clés nommées

Tableaux multidimensionnels

Un tableau qui contient d'autres tableaux comme éléments.

<?php
$etudiants = [
    [
        "nom" => "Martin",
        "notes" => [15, 12, 18]
    ],
    [
        "nom" => "Bernard",
        "notes" => [10, 14, 16]
    ]
];

// Accès aux données
echo $etudiants[0]["nom"]; // Affiche: Martin
echo $etudiants[1]["notes"][0]; // Affiche: 10
?>

Fonctions utiles pour les tableaux

Fonctions de base

Fonction Description Exemple
count() Nombre d'éléments count($fruits) → 3
array_push() Ajoute un élément à la fin array_push($fruits, "Kiwi")
array_pop() Supprime le dernier élément array_pop($fruits)
array_merge() Fusionne des tableaux array_merge($tab1, $tab2)
in_array() Vérifie si une valeur existe in_array("Pomme", $fruits)
array_search() Recherche une valeur array_search("Banane", $fruits)

Parcourir un tableau

<?php
$fruits = ["Pomme", "Banane", "Orange"];

// Avec for
for ($i = 0; $i < count($fruits); $i++) {
    echo $fruits[$i] . "<br>";
}

// Avec foreach (recommandé)
foreach ($fruits as $fruit) {
    echo $fruit . "<br>";
}

// Avec foreach et clés (tableau associatif)
foreach ($personne as $cle => $valeur) {
    echo $cle . ": " . $valeur . "<br>";
}
?>

Comparaison des types de tableaux

Type Clés Utilisation Exemple
Indexé Numériques (0, 1, 2...) Listes ordonnées ["Pomme", "Banane", "Orange"]
Associatif Chaînes de caractères Données structurées ["nom" => "Dupont", "age" => 25]
Multidimensionnel Mixte Données complexes [["nom" => "A", "notes" => [12,15]]]

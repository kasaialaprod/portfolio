Structure d'un programme C#

using, namespace, classe, méthode Main

Un programme console C# classique est composé de directives using, d'un namespace, d'une class et d'une méthode Main servant de point d'entrée au programme [web:16][web:13].

// Exemple minimal
using System;

namespace AstroBot.Core
{
    public class Program
    {
        public static void Main(string[] args)
        {
            Console.WriteLine("Boot système Astrobot...");
        }
    }
}
  • using System; : donne accès à Console et autres types du namespace System [web:16].
  • namespace AstroBot.Core : espace logique pour organiser tes classes.
  • public class Program : vaisseau principal contenant le code exécutable.
  • static void Main(string[] args) : salle de contrôle appelée au démarrage du programme [web:16].

Classes et modificateurs d'accès

public, private et organisation du code

Une classe regroupe des données et des comportements, avec des modificateurs d'accès pour contrôler ce qui est visible à l'extérieur [web:21][web:24].

public class Drone
{
    // Champ privé (interne à la classe)
    private int energie;

    // Propriété publique (visible de l'extérieur)
    public int Energie
    {
        get { return energie; }
        set { energie = value; }
    }

    // Méthode publique
    public void Boot()
    {
        Console.WriteLine("Drone en ligne.");
    }
}
  • public : accessible depuis l'extérieur de la classe/du namespace [web:21].
  • private : accessible uniquement à l'intérieur de la classe (valeur par défaut pour les champs) [web:21][web:30].
  • Bon réflexe : garder les données sensibles en private et exposer seulement ce qui est nécessaire via des membres publics.

Méthodes

Signature, type de retour, paramètres

Une méthode est une fonction attachée à une classe, décrite par un modificateur, un type de retour, un nom, d'éventuels paramètres et un corps de code [web:16].

public int Additionner(int a, int b)
{
    int resultat = a + b;
    return resultat;
}
  • public int Additionner(int a, int b) : méthode publique qui renvoie un int.
  • void signifie que la méthode ne renvoie aucune valeur.

Déclaration de variables

Simple, initialisation, déclarations multiples
  • Déclaration simple : int score;
  • Déclaration + initialisation : int score = 0; [web:14].
  • Déclaration multiple : int x = 1, y = 2, z = 3;

Le type est toujours indiqué avant le nom ; l'inférence avec var se fera plus tard.

Types de données

Numériques, booléens, caractères, chaînes
  • int : entier 32 bits, type le plus courant pour compter [web:14].
  • double : réel en double précision.
  • float : réel simple précision (suffixe f).
  • bool : true ou false.
  • char : caractère unique, entre apostrophes ('A').
  • string : texte, entre guillemets ("Hello") [web:14].

On utilise ces types pour exprimer des concepts mathématiques simples, comme le calcul d'un score, d'une moyenne ou d'un pourcentage de batterie de ton astrobot.

Opérateurs arithmétiques

+, -, *, /, %
  • + : addition.
  • - : soustraction.
  • * : multiplication.
  • / : division.
  • % : modulo (reste de la division) [web:28].
int a = 10;
int b = 3;
int somme = a + b;      // 13
int reste = a % b;      // 1

Incrémentation, décrémentation et affectation

++, --, +=, -=, *=, /=, %=
  • x++ / ++x : ajoute 1 à x.
  • x-- / --x : enlève 1 à x [web:22].
int n = 5;
n++;      // n = 6
++n;      // n = 7

Préfixe ++x : on modifie puis on utilise la nouvelle valeur ; postfixe x++ : on utilise l'ancienne valeur puis on modifie [web:22][web:25].

  • x += 2; équivaut à x = x + 2;
  • x -= 1;, x *= 3;, x /= 2;, x %= 10; [web:28].

Entrées utilisateur et conversions

Console.ReadLine, int.Parse, Convert

Pour lire une entrée clavier, on utilise Console.ReadLine() qui renvoie une chaîne de caractères [web:14].

Console.Write("Entrer un niveau d'énergie : ");
string input = Console.ReadLine();

int energie = int.Parse(input);          // suppose une entrée valide
// ou
int energie2 = Convert.ToInt32(input);   // autre méthode de conversion

Pour la robustesse, on peut utiliser int.TryParse afin d'éviter les exceptions si l'entrée n'est pas un nombre.

Chaînes de caractères

Échappement, verbatim, concaténation, interpolation
  • Caractère d'échappement \n : saut de ligne.
  • \t : tabulation horizontale.
  • \\ : barre oblique inversée.
  • \" : guillemet dans une chaîne [web:14].
string log = "Boot Astrobot\\Core\\v1.0\\n\\tSystème prêt.";

Une chaîne verbatim commence par @ et conserve les backslashes et retours à la ligne tels quels.

string chemin = @"C:\Astrobot\logs\core.log";

Concaténation simple :

string nom = "Astrobot";
string msg = "Salut " + nom + " !";

Interpolation de chaînes (recommandée) :

int niveau = 42;
string status = $"[Core] Niveau d'énergie : {niveau}%";

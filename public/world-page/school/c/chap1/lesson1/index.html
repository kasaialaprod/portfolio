> Découverte du langage C
Le langage des OS, des machines, des réseaux…

1. Structure d’un programme C (le squelette)

  • Tout programme démarre à main()
  • Utilise #include pour importer des bibliothèques
  • Chaque ligne d’instruction se termine par “;”
#include <stdio.h> // Pour l’affichage

int main() {
  printf("Bienvenue dans C !\n");
  return 0;
}
$ gcc prog.c -o prog
$ ./prog
Bienvenue dans C !

2. Déclaration de variables

  • Chaque variable doit être déclarée et typée : int, float, char…
  • Les variables peuvent être initialisées directement ou plus tard
int age = 21;
float taille = 1.76;
char lettre = 'A';
char pseudo[16] = "cyber_link";

3. Types de données principaux

  • int : entier
  • float, double : nombre à virgule (float peut stocker jusqu'à 7 chiffres significatifs, double environ 15)
  • char : caractère unique
  • char [ ] : chaîne de caractères (tableau de char)

4. Formatage de la sortie (printf)

  • printf affiche texte, variables, calculs…
  • Utilise des spécificateurs pour chaque type :
  • %d (entier), %f (décimal), %c (char), %s (string, tableau de char)
  • \n pour nouvelle ligne
  • Nota Bene %f affiche 6 décimales par défaut. Pour en limiter le nombre, utiliser %.1f (1 décimale), %.2f (2 décimales)…
int points = 42;
float vie = 75.5;
printf("Points : %d, Vie : %.1f\n", points, vie);
char pseudo[10] = "Link";
printf("Pseudo : %s\n", pseudo);
Points : 42, Vie : 75.5
Pseudo : Link

5. Mathématiques de base

  • Opérateurs : +, -, *, /, %
  • Attention : division entière : int a = 7 / 2; renvoie 3 (pas 3.5)
  • Pour accès aux fonctions mathématiques avancées (sqrt, pow…), inclure math.h
#include <math.h>
double racine = sqrt(81);
double puissance = pow(2, 8);
printf("√81 = %.0f, 2^8 = %.0f\n", racine, puissance);
√81 = 9, 2^8 = 256

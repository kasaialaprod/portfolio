Fonctions et pointeurs

1. Fonction : définition & appel

  • Une fonction permet de réutiliser et organiser ton code
  • Elle possède un type de retour, un nom, des paramètres éventuels
// "int" = type de retour

int somme(int a, int b) {
return a + b;
}

int main() {
  int x = somme(4, 7);
  printf("Résultat : %d\n", x);
  return 0;
}
Résultat : 11

2. Paramètres et retour de fonction

  • Les paramètres transmettent des valeurs à la fonction
  • Le mot-clé return renvoie la valeur finale au programme
  • Le type avant le nom de la fonction = le type retourné (void si rien)
// Fonction sans retour, avec paramètres :

void bonjour(char nom[]) {
  printf("Bienvenue %s\n", nom);
}
// Fonction avec retour et paramètre :

float carre(float x) {
  return x * x; // retourne x au carré
}

3. Les pointeurs : adresser la mémoire

  • Un pointeur stocke l’adresse d’une variable en mémoire.
  • Déclaration : int *ptr; (ptr pointe vers un int)
  • Utilisation :
    • &x = adresse de x
    • *ptr = valeur “pointée”, obtenue via l'adresse
int x = 10;
int *p;
p = &x; // p pointe vers x
printf("%d\n", *p); // affiche la valeur pointée (10)

Pour illustrer le principe de poiteur avec un exemple un peut plus avancé :

Sans pointeur :
void ajouteUn(int n) {
  n = n + 1; // ne change que la copie
}

int main() {
  int score = 5;
  ajouteUn(score);
  printf("%d\n", score); // 5, pas 6
}

La fonction ne modifie pas la variable score car elle travaille sur une copie de score dans un nouveau int appelé n. Cela signifie que seul n est modifié, pas score.


Avec pointeur :
void ajouteUn(int *n) {
  *n = *n + 1; // modifie la vraie variable
}

int main() {
  int x = 5;
  ajouteUn(&x);
  printf("%d\n", x);
  return 0;
}

En utilisant un pointeur, la fonction peut accéder et modifier la variable originale en mémoire. En passant l'adresse de score à la fonction, elle peut modifier directement la valeur stockée à cette adresse.

4. Utilité des pointeurs : modification & passage par référence

  • Grâce aux pointeurs, tu peux modifier une variable dans une fonction
  • On envoie l'adresse à la fonction, et elle peut modifier “en place”
void ajouteUn(int *n) {
  *n = *n + 1; // modifie la vraie variable
}

int main() {
  int score = 5;
  ajouteUn(&score);
  printf("%d\n", score); // affiche 6
}

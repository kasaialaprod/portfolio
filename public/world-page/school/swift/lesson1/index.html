Leçon 1 : Setup & Premiers pas avec VS Code

1. Installer Swift sur macOS

Swift est déjà installé si tu as Xcode, mais on peut l'installer via Homebrew pour l'utiliser en ligne de commande.

Vérifier si Swift est installé

$ swift --version
swift-driver version: 1.87.3 Apple Swift version 5.9.2

Si tu vois une version, c'est bon. Sinon, installe via Homebrew :

$ brew install swift

2. Configurer VS Code pour Swift

2.1 Installer l'extension Swift

Dans VS Code, va dans l'onglet Extensions et installe :

  • Extension Swift (par Swift Server Work Group)

2.2 Créer un dossier de projet

$ mkdir swift-apprentissage
$ cd swift-apprentissage
$ code .

VS Code s'ouvre dans ton dossier.

3. Premier fichier Swift

3.1 Créer main.swift

Dans VS Code, crée un fichier de>main.swift à la racine de ton projet.

3.2 Hello World

print("Hello, Swift depuis VS Code!")

3.3 Exécuter le code

Ouvre le terminal intégré de VS Code (de>Ctrl + ` ou de>Cmd + `) et tape :

$ swift main.swift
Hello, Swift depuis VS Code!

✨ Félicitations, ton premier programme Swift fonctionne !

4. Compiler vs Interpréter

4.1 Mode "interprété" (rapide pour tester)

$ swift main.swift

Execute directement le fichier sans créer d'exécutable.

4.2 Mode "compilé" (créer un exécutable)

$ swiftc main.swift -o monprogramme
$ ./monprogramme
Hello, Swift depuis VS Code!

de>swiftc compile ton code en un fichier binaire exécutable (de>monprogramme).

5. Structure d'un fichier Swift de base

// Ceci est un commentaire

// Import d'un module (optionnel pour du Swift basique)
import Foundation

// Point d'entrée : le code est exécuté ligne par ligne
print("Bienvenue dans Swift")

let nom = "Marcus"
print("Hello, \(nom)!")
  • de>// pour les commentaires
  • de>import Foundation donne accès à des utilitaires (dates, strings avancés, etc.)
  • de>print() affiche dans la console
  • de>let déclare une constante
  • de>\(nom) = interpolation de variable dans une string

6. Swift REPL (mode interactif)

Tu peux aussi lancer Swift en mode interactif (comme Python) :

$ swift
Welcome to Apple Swift version 5.9.2
1> let x = 42
2> print(x)
42
3> :quit

Pratique pour tester des lignes de code rapidement.

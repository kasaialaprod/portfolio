Leçon 5 : SwiftUI — Premier projet d'interface graphique

1. C'est quoi SwiftUI ?

SwiftUI est le framework moderne d'Apple pour créer des interfaces utilisateur de façon déclarative.

  • ✅ Déclaratif : tu décris ce que tu veux, pas comment le faire
  • ✅ Multiplateforme : iOS, macOS, watchOS, tvOS avec le même code
  • ✅ Preview en temps réel : vois tes changements instantanément dans Xcode
  • ✅ Syntaxe moderne : utilise les closures, property wrappers (@State, @Binding...)
À partir de cette leçon, tu dois utiliser Xcode (pas VS Code). SwiftUI nécessite le simulateur et les outils de preview d'Xcode.

2. Créer un projet SwiftUI dans Xcode

Étapes

  1. Lance Xcode
  2. Clique sur "Create New Project"
  3. Choisis iOS → App
  4. Remplis :
    • Product Name : MonAppSwiftUI
    • Interface : SwiftUI
    • Language : Swift
  5. Clique sur Next puis Create

Xcode génère automatiquement un fichier de>ContentView.swift avec une vue de base.

3. Structure d'une vue SwiftUI

Fichier ContentView.swift (généré par défaut)

import SwiftUI struct ContentView: View { var body: some View { Text("Hello, World!") } } #Preview { ContentView() }

Décryptage

  • de>struct ContentView: View : ta vue conforme au protocole de>View
  • de>var body: some View : propriété obligatoire qui décrit le contenu de la vue
  • de>Text("Hello, World!") : un composant texte
  • de>#Preview : macro pour afficher un aperçu en temps réel dans Xcode

4. Views de base

4.1 Text (texte)

Text("Bienvenue dans SwiftUI") .font(.title) .foregroundColor(.blue) .bold()

4.2 Image (image système SF Symbols)

Image(systemName: "star.fill") .resizable() .frame(width: 50, height: 50) .foregroundColor(.yellow)

4.3 Button (bouton)

Button("Clique-moi") { print("Bouton cliqué") } .buttonStyle(.borderedProminent)

4.4 TextField (champ de texte)

@State private var nom = "" TextField("Entre ton nom", text: $nom) .textFieldStyle(.roundedBorder) .padding()
de>@State crée une variable réactive : quand elle change, la vue se rafraîchit automatiquement.

5. Layout : VStack, HStack, ZStack

5.1 VStack (vertical)

VStack { Text("Ligne 1") Text("Ligne 2") Text("Ligne 3") }

5.2 HStack (horizontal)

HStack { Image(systemName: "star.fill") Text("Marcus") Spacer() Text("Niveau 10") }

5.3 ZStack (superposition)

ZStack { Circle() .fill(Color.blue) .frame(width: 100, height: 100) Text("42") .foregroundColor(.white) .font(.largeTitle) }

5.4 Spacing & Padding

VStack(spacing: 20) { Text("Haut") Text("Bas") } .padding() // ajoute de l'espace autour

6. @State : gérer l'état local

de>@State crée une variable réactive : quand elle change, SwiftUI redessine automatiquement la vue.

Exemple : compteur

struct ContentView: View { @State private var compteur = 0 var body: some View { VStack(spacing: 20) { Text("Compteur : \(compteur)") .font(.largeTitle) Button("Incrémenter") { compteur += 1 } .buttonStyle(.borderedProminent) } .padding() } }

À chaque clic sur le bouton, de>compteur augmente et le de>Text se met à jour automatiquement.

7. Mini app complète : Salutation

struct ContentView: View { @State private var nom = "" var body: some View { VStack(spacing: 25) { Text("Bienvenue") .font(.largeTitle) .bold() TextField("Entre ton pseudo", text: $nom) .textFieldStyle(.roundedBorder) .padding(.horizontal) if !nom.isEmpty { Text("Salut, \(nom) !") .font(.title2) .foregroundColor(.blue) } Spacer() } .padding() } }

Fonctionnement :

  • L'utilisateur tape son pseudo dans le de>TextField
  • La variable de>nom se met à jour en temps réel
  • Si de>nom n'est pas vide, un message de salutation apparaît

8. Modifiers (modificateurs de vue)

Les modifiers changent l'apparence et le comportement des vues. Ils se chaînent.

Text("SwiftUI") .font(.largeTitle) .foregroundColor(.blue) .padding() .background(Color.yellow) .cornerRadius(10) .shadow(radius: 5)

Modifiers courants

  • de>.font() : taille et style de police
  • de>.foregroundColor() : couleur du texte
  • de>.padding() : espacement intérieur
  • de>.background() : couleur/image de fond
  • de>.cornerRadius() : coins arrondis
  • de>.shadow() : ombre portée
  • de>.frame(width:height:) : taille fixe

9. Lancer l'app dans le simulateur

Étapes

1. En haut de Xcode, choisis un simulateur (ex : iPhone 15 Pro)
2. Clique sur le bouton Play ▶ (ou de>Cmd + R)
3. Le simulateur iOS se lance et affiche ton app
4. Modifie ton code → sauvegarde → l'app se recharge automatiquement

10. Preview en temps réel (Canvas)

Dans Xcode, active le Canvas (panneau de droite) pour voir ton interface se mettre à jour en direct pendant que tu codes.

Activer le Canvas

  • Menu : Editor → Canvas
  • Ou : de>Cmd + Option + Enter

Dès que tu modifies ton code, le Canvas rafraîchit l'aperçu automatiquement.

11. Exercice pratique : Carte de profil

Crée une carte de profil avec photo, nom, et bouton.

struct ContentView: View { var body: some View { VStack(spacing: 20) { // Avatar Image(systemName: "person.circle.fill") .resizable() .frame(width: 120, height: 120) .foregroundColor(.blue) // Nom Text("Marcus Holloway") .font(.title) .bold() // Rôle Text("Hacker DedSec") .font(.subheadline) .foregroundColor(.secondary) // Stats HStack(spacing: 30) { VStack { Text("250") .font(.title2) .bold() Text("Missions") .font(.caption) .foregroundColor(.secondary) } VStack { Text("10") .font(.title2) .bold() Text("Niveau") .font(.caption) .foregroundColor(.secondary) } VStack { Text("5.2k") .font(.title2) .bold() Text("XP") .font(.caption) .foregroundColor(.secondary) } } .padding() // Bouton Button("Voir le profil") { print("Profil ouvert") } .buttonStyle(.borderedProminent) .controlSize(.large) } .padding() } }

Lance cette app dans le simulateur pour voir le résultat !

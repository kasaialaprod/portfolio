Propriétés CSS du Texte

font-family

font-family: Arial, sans-serif;

Définit la famille de police

font-size

font-size: 16px;

Définit la taille de la police

font-weight

font-weight: bold;

Contrôle l'épaisseur du texte

line-height

line-height: 1.5;

Espacement entre les lignes

text-align

text-align: center;

Alignement horizontal du texte

letter-spacing

letter-spacing: 1px;

Espacement entre les caractères

Intégration avec Google Fonts

Méthode d'import

@import url('https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap');

.ma-classe {
    font-family: 'Roboto', sans-serif;
    font-weight: 700;
}

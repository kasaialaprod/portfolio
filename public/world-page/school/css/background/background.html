Bonnes pratiques

Performance

  • Préférez les dégradés CSS aux images

  • Utilisez le format WebP quand possible

Accessibilité

  • Contraste suffisant texte/background

  • Fournissez des alternatives

Maintenance

  • Utilisez des variables CSS
  • Organisez vos backgrounds
  • Documentez les combinaisons complexes

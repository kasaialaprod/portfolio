Sélecteurs de Base

Les fondamentaux du ciblage CSS

Sélecteur d'Élément

p { color: blue; }

Cible tous les éléments <p>

Ce paragraphe est ciblé

Sélecteur de Classe

.ma-classe { color: red; }

Cible les éléments avec class="ma-classe"

Ce texte a une classe

Sélecteur d'ID

#mon-id { color: green; }

Cible l'élément avec id="mon-id"

Ce texte a un ID

Sélecteur Universel

* { margin: 0; }

Cible tous les éléments de la page

Sélecteur Groupé

h1, h2, h3 { font-family: Arial; }

Cible plusieurs sélecteurs en même temps

Pseudo-éléments

::before

.element::before {
content: "★ ";
color: gold;
}

Élément avec contenu avant

::after

.element::after {
content: " !";
color: red;
}

Élément avec contenu après

::first-letter

p::first-letter {
font-size: 2em;
color: blue;
}

Première lettre stylisée

::first-line

p::first-line {
font-weight: bold;
color: green;
}

Ceci est la première ligne qui sera stylisée différemment du reste du paragraphe. Le reste du texte conserve son apparence normale.

::selection

::selection {
background: yellow;
color: black;
}

Sélectionne ce texte pour voir l'effet

Sélecteurs d'Attributs

[attribut]

[disabled] { opacity: 0.5; }

Cible les éléments avec l'attribut

[attribut=valeur]

[type="password"] {
font-family: monospace;
}

Cible avec valeur exacte

[attribut^=valeur]

[href^="https"] {
color: green;
}

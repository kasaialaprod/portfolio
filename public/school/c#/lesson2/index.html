Booléens & opérateurs de comparaison

true / false pour piloter le code

Un booléen est un type qui ne vaut que true ou false et sert à prendre des décisions [web:33].

  • == égal à
  • != différent de
  • >, <, >=, <= comparaisons d'ordre [web:33][web:36].
int age = 18;
bool majeur = age >= 18;   // true
bool special = age == 42;  // false

if / else

Exécuter un bloc seulement si la condition est vraie
int age = 18;

if (age >= 18)
{
    Console.WriteLine("Accès autorisé au pont principal.");
}
else
{
    Console.WriteLine("Accès refusé, niveau trop bas.");
}

La condition entre parenthèses doit être un booléen ; le bloc else est optionnel [web:41].

switch

Choisir parmi plusieurs cas
int mode = 2;

switch (mode)
{
    case 1:
        Console.WriteLine("Mode veille.");
        break;
    case 2:
        Console.WriteLine("Mode exploration.");
        break;
    default:
        Console.WriteLine("Mode inconnu.");
        break;
}

switch est pratique quand il y a plusieurs valeurs possibles sur une même variable [web:38].

Opérateurs logiques & ternaire

Combiner des conditions
  • && : ET logique (les deux conditions vraies).
  • || : OU logique (au moins une vraie).
  • ! : NON logique (inverse un booléen) [web:39][web:45].
bool batterieOk = niveauBatterie >= 30;
bool liaisonOk  = signal >= 2;

if (batterieOk && liaisonOk)
{
    Console.WriteLine("Décollage autorisé.");
}

Opérateur ternaire : condition ? valeurSiVrai : valeurSiFaux [web:49].

string statut = (age >= 18) ? "Majeur" : "Mineur";

Boucles for & while

Répéter une action tant qu'une condition est vraie
// Boucle for
for (int i = 0; i < 5; i++)
{
    Console.WriteLine($"Scan #{i}");
}
// Boucle while
int energie = 100;
while (energie > 0)
{
    Console.WriteLine($"Énergie : {energie}%");
    energie -= 10;
}

break sort de la boucle immédiatement ; continue saute à l'itération suivante [web:41].

for (int i = 0; i < 10; i++)
{
    if (i == 3) continue; // saute 3
    if (i == 8) break;    // sort
    Console.WriteLine(i);
}

Méthodes, paramètres et retours

Déclarer, appeler, passer des données et renvoyer un résultat

Une méthode se déclare avec un type de retour, un nom et des paramètres éventuels [web:49].

public int CalculerEnergieRestante(int capaciteMax, int conso)
{
    int restante = capaciteMax - conso;
    return restante;
}

Ici, la méthode prend deux paramètres int et renvoie un int via return.

// Appel de méthode
int max = 100;
int consommee = 35;

int reste = CalculerEnergieRestante(max, consommee);
Console.WriteLine($"Énergie restante : {reste}%");

On parle de paramètres dans la déclaration et d'arguments lors de l'appel.

Structures de contrôle et boucles

Conditions Boucles Comparaison

Introduction aux structures de contrôle

Les structures de contrôle permettent de modifier le flux d'exécution d'un programme en fonction de conditions ou de répéter des blocs de code.

Flux d'exécution d'un programme

[Début] → [Condition] → [Action si vrai] → [Action si faux] → [Fin]

Les structures conditionnelles

La condition IF

Exécute un bloc de code si une condition est vraie.

<?php
$age = 18;

if ($age >= 18) {
    echo "Vous êtes majeur";
}
?>

IF...ELSE

Exécute un bloc si la condition est vraie, un autre si elle est fausse.

<?php
$temperature = 25;

if ($temperature > 30) {
    echo "Il fait chaud";
} else {
    echo "Il fait bon";
}
?>

IF...ELSEIF...ELSE

Teste plusieurs conditions successivement.

<?php
$note = 14;

if ($note >= 16) {
    echo "Très bien";
} elseif ($note >= 14) {
    echo "Bien";
} elseif ($note >= 12) {
    echo "Assez bien";
} else {
    echo "Insuffisant";
}
?>

SWITCH

Alternative à une série de conditions IF, utile pour comparer une variable à plusieurs valeurs.

<?php
$jour = "lundi";

switch ($jour) {
    case "lundi":
        echo "Début de semaine";
        break;
    case "vendredi":
        echo "Bientôt le weekend";
        break;
    default:
        echo "Jour ordinaire";
}
?>

Attention: N'oubliez pas le break dans un switch, sinon l'exécution continuera aux cas suivants!

Les boucles

Boucle WHILE

Répète un bloc tant qu'une condition est vraie.

<?php
$i = 1;

while ($i <= 5) {
    echo "Tour numéro: " . $i . "<br>";
    $i++;
}
?>

Boucle DO...WHILE

Similaire à WHILE, mais exécute le bloc au moins une fois.

<?php
$i = 10;

do {
    echo "Valeur: " . $i . "<br>";
    $i++;
} while ($i <= 5);
?>

Boucle FOR

Boucle avec initialisation, condition et incrémentation regroupées.

<?php
for ($i = 1; $i <= 5; $i++) {
    echo "Itération: " . $i . "<br>";
}
?>

Boucle FOREACH

Spécialement conçue pour parcourir les tableaux.

<?php
$fruits = array("pomme", "banane", "orange");

foreach ($fruits as $fruit) {
    echo $fruit . "<br>";
}

// Avec les clés
$personne = array(
    "nom" => "Dupont",
    "prenom" => "Jean",
    "age" => 25
);

foreach ($personne as $cle => $valeur) {
    echo $cle . ": " . $valeur . "<br>";
}
?>

Comparaison des boucles

Type de boucle Utilisation Avantages
WHILE Quand le nombre d'itérations est inconnu Flexible, condition évaluée avant
DO...WHILE Quand on veut exécuter au moins une fois Garantit une exécution minimum
FOR Quand le nombre d'itérations est connu Structure compacte et lisible
FOREACH Pour parcourir des tableaux Simple et optimisé pour les tableaux

Interrompre les boucles

BREAK

Sort immédiatement d'une boucle.

<?php
for ($i = 1; $i <= 10; $i++) {
    if ($i == 5) {
        break; // Sort de la boucle quand i=5
    }
    echo $i . "<br>";
}
?>

CONTINUE

Passe à l'itération suivante sans exécuter le reste du bloc.

<?php
for ($i = 1; $i <= 5; $i++) {
    if ($i == 3) {
        continue; // Saute l'itération quand i=3
    }
    echo $i . "<br>";
}
?>

Démonstration interactive

Générateur de table de multiplication

Ce code génère la table de multiplication d'un nombre:

<?php
$nombre = num;
echo "Table de multiplication de " . $nombre . ":<br>";

for ($i = 1; $i <= 10; $i++) {
    $resultat = $nombre * $i;
    echo $nombre . " × " . $i . " = " . $resultat . "<br>";
}
?>

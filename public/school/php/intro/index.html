Histoire de PHP

PHP (Personal Home Page, maintenant PHP: Hypertext Preprocessor) a été créé en 1994 par Rasmus Lerdorf. À l'origine, c'était un ensemble de scripts Perl pour suivre les visites sur son CV en ligne.

Saviez-vous? Le premier nom de PHP était "Personal Home Page Tools".

En 1995, Lerdorf a publié la première version publique, PHP/FI (Forms Interpreter). Cette version incluait déjà certaines fonctionnalités de base que nous connaissons aujourd'hui.

PHP a évolué au fil des versions :

  • PHP 3 (1998) : Première version ressemblant à PHP moderne
  • PHP 4 (2000) : Introduction du moteur Zend
  • PHP 5 (2004) : Modèle objet amélioré
  • PHP 7 (2015) : Performances considérablement améliorées
  • PHP 8 (2020) : JIT compiler et nouvelles fonctionnalités

Les Variables en PHP

En PHP, les variables sont des conteneurs pour stocker des données. Elles sont identifiées par le signe dollar $.

Déclaration de variables

<?php
$nom = "Jean";
$age = 25;
$est_etudiant = true;
?>

Types de données

PHP supporte plusieurs types de données :

  • String : Chaîne de caractères (texte)
  • Integer : Nombre entier
  • Float : Nombre à virgule flottante
  • Boolean : Vrai ou faux
  • Array : Tableau
  • Object : Objet
  • NULL : Valeur nulle

Affichage de variables

<?php
echo $nom; // Affiche: Jean
echo "Bonjour " . $nom; // Affiche: Bonjour Jean
echo "Vous avez " . $age . " ans"; // Affiche: Vous avez 25 ans
?>

Assignations de variables

<?php
$nom = 'Jean'; // La variable $nom prend la valeur 'Jean'
$age == 25; // La valeur $age est égale à 25
$age === 25; // La valeur $age es strictement égale à 25 (en parle du type de donnée)

echo "Prénom: " . $nom . " Âge: " . $age . "ans"; // Affiche: "Prénom: Jean Âge: 25ans"
?>

Calculs Simples en PHP

PHP supporte tous les opérateurs arithmétiques de base.

Opérateurs arithmétiques

<?php
$a = 10;
$b = 3;

$somme = $a + $b; // 13
$difference = $a - $b; // 7
$produit = $a * $b; // 30
$quotient = $a / $b; // 3.333...
$reste = $a % $b; // 1
$puissance = $a ** $b; // 1000 (10^3)
?>

Exemple pratique

<?php
$prix_ht = 50;
$tva = 0.2; // 20%
$prix_ttc = $prix_ht * (1 + $tva);

echo "Prix TTC: " . $prix_ttc . " €"; // Affiche: Prix TTC: 60 €
?>

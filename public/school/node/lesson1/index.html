> NODE.JS 101

// Leçon 1 : Installation & premier script côté serveur

1. C’est quoi Node.js ?

Node.js est un environnement qui permet d’exécuter du JavaScript en dehors du navigateur, directement dans le terminal, sur ton système.

Il est construit sur le moteur V8 de Chrome et est pensé pour créer des scripts, des outils en ligne de commande, des APIs et des serveurs web rapides.
NOTA BENE: V8 = le "cerveau" qui comprend et exécute le JavaScript.

INFO
$ node script.js // exécuter un fichier JavaScript côté backend
$ npm install ... // gérer les dépendances de ton projet

2. Installation de Node.js

On installe la version LTS (support long terme), recommandée pour apprendre et pour les projets stables.

Windows

  • Va sur le site officiel : https://nodejs.org
  • Télécharge l’installateur LTS (.msi) et lance-le.
  • Suivre “Next, Next…” en gardant npm coché, puis “Install”.

macOS

  • Soit tu télécharges le .pkg sur https://nodejs.org, soit tu fais brew install node si tu as Homebrew.

Linux (Debian/Ubuntu)

  • Dans le terminal : sudo apt update puis sudo apt install nodejs npm.

3. Vérifier que Node.js est bien installé

Ouvre un terminal (PowerShell / CMD / Terminal) et tape :

$ node -v
v18.x.x // numéro de version Node.js

$ npm -v
10.x.x // numéro de version npm

Si tu vois des numéros de version, l’installation est ok.

4. Ton premier programme Node.js

1. Crée un dossier où tu veux, par exemple node-playground.

2. Dans ce dossier, crée un fichier app.js avec ce contenu :

// app.js
console.log("DedSec Node.js online.");
console.log("Hello from the backend.");

3. Dans le terminal, place-toi dans ce dossier :

$ cd chemin/vers/node-playground
$ node app.js

Tu dois voir le message s’afficher dans le terminal : c’est Node qui exécute ton JavaScript côté machine.

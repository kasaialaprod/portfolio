Introduction au JAVASCRIPT

Accepte la mission, jeune développeur, et découvre la magie du code côté navigateur !

Bienvenue, aventurier !
Dans ce royaume du web, JavaScript est la baguette magique qui donne vie à tes pages. Prêt à décocher ton premier “script” ?

Qu’est-ce que JavaScript ?

  • Un langage de programmation indispensable pour ajouter de l’interactivité à une page web.
  • Il permet de réagir aux actions du joueur (clics, déplacements, appuis clavier...)
  • Sans JS, une page reste statique : avec JS, c’est une aventure !

Anecdote Hylienne : Même les coffres des temples s’ouvriraient grâce à une fonction JavaScript cachée...

La déclaration de variables — “Découvre tes objets mystiques”

En JavaScript, tu utilises 3 mots magiques: let, const et var pour créer tes objets (variables) :

// Épée que tu peux changer plus tard
let epee = "Lame d'Hylia";


// Bouclier sacré que tu ne changeras pas
const bouclier = "Bouclier Hylien";


// Méthode ancienne (évite-la), valable mais poussiéreuse
var potion = "Remède ancien";
  • let : pour une variable qui peut évoluer
  • const : pour une variable qui ne change jamais
  • var : ancêtre, à éviter pour éviter les bugs mystérieux

Chaque variable est comme une place dans ton inventaire !

Sémantique de base — “Parle comme un Sage”

Les noms de variables parlent au Héros : choisis des noms clairs et logiques, évite les malédictions du “x” ou “y” sans explication.

// Mauvais exemple
let x = 7;

// Bon exemple
let nombreDeRubis = 7;

JS ne tolère pas les accents ni les espaces dans les noms, mais accepte les lettres, chiffres et "_". Convention : commence par une minuscule et utilise le camelCase (ex. : nombreDeCœurs). Ainsi, tu laisses une trace claire pour les autres aventuriers qui liront ton code !

Mini Quête : Manipuler le DOM (“le monde à façonner”)

Le DOM (= Document Object Model) est la carte magique de ta page : JS te permet d’en modifier chaque brique !

// Afficher un message dans une boîte secrète

let text = document.getElementById("secretBox");
text.textContent = 'TU NE ME VOIS PLUS !!!';
  • let text : crée une variable pour stocker un élément HTML
  • document.getElementById(): pour trouver un coffre (élément HTML) précis
  • text.textContent: pour modifier le contenu du coffre

Opérations & Données

Nouvelle aventure : déverrouille le pouvoir des opérations magiques, découvre les secrets des chaînes, des types et des énigmes logiques !
Dans cette épreuve, tu manieras toute la magie du code : comparer des reliques, additionner des rubis, déverrouiller des coffres avec les énigmes logiques… Et apprendre à demander son nom au voyageur qui visite la page !
Quête 1 : Les opérations de comparaison
  • Utilise == pour comparer mais === pour être sûr que la relique est 100 % identique (type compris) !

  • != et !== pour vérifier la différence (de valeur ou de type)

  • >, <, >=, <= pour les batailles de puissance
let rubisHero = 40;
let rubisPNJ = "40";

console.log(rubisHero == rubisPNJ); // true
console.log(rubisHero === rubisPNJ); // false
console.log(rubisHero < 2); // false console.log(rubisHero != 30); // true
Quête 2 : Les opérations mathématiques & logiques
  • Addition (+), soustraction (-), multiplication (*), division (/), reste (%)

  • Logique : && (ET), || (OU), ! (NON)
let coeurs = 7;
let stamina = 4;


console.log(coeurs + stamina); // 11

console.log(coeurs > 5 && stamina > 3); // true && true = true

console.log(coeurs < 5 || stamina > 3); // false || true = true
Quête 3 : Manipulation des chaînes de caractères
  • Concaténation : assemble des mots avec +
  • Longueur : .length
  • Transformation : .toUpperCase(), .toLowerCase()
let hero = "Link";

console.log("Nom du héros : " + hero);
console.log(hero.length); // 4
console.log(hero.toUpperCase()); // "LINK"
Quête 4 : Assignations et entrées utilisateur
  • Assignation : utiliser = pour donner une valeur
  • Entrées utilisateur simples : prompt()
  • Entrées complexes : via des formulaires HTML + JS
event.preventDefault(); // Annule le reload de la page

let nom = document.getElementById('nom').value;

let messageDeBienvenue = document.getElementById('entréeUtilisateur');

messageDeBienvenue.textContent = "Bienvenu " + nom;
Quête 5 : Les types de données
  • Number : nombre (entier ou décimal)
  • String : chaîne de caractères
  • Boolean : vrai ou faux
  • Array : liste d'objets
  • Object : caractéristiques d’un objet (ex : héros avec force, vie…)
let vie = 18; // Number

let nom = "Zelda"; // String

let estHylien = true; // Boolean

let inventaire = ["épée", "potion", "rubis"]; // Array

let heros = { nom: "Link", coeurs: 9, sagesse: 14 }; // Object

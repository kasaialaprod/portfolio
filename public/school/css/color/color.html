Unités Relatives

EM - Element EM (1.2em)
% - Pourcentage (50%)
VW - Viewport Width (50vw)
VH - Viewport Height (50vh)

Comparaison REM vs EM

REM (Relative au root)

Niveau 1 - 1.5rem
Niveau 2 - 1.5rem
Niveau 3 - 1.5rem

EM (Relative au parent)

Niveau 1 - 1.5em
Niveau 2 - 1.5em
Niveau 3 - 1.5em

Media Queries

/* Mobile First */
.container {
    padding: 1rem;
}

/* Tablette */
@media (min-width: 768px) {
    .container {
        padding: 2rem;
        max-width: 720px;
    }
}

/* Desktop */
@media (min-width: 1024px) {
    .container {
        padding: 3rem;
        max-width: 1200px;
    }
}

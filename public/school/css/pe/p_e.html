Terminologie

  • Parent : Élément direct au-dessus
  • Enfant : Élément direct en-dessous
  • Ancêtre : Tous les éléments au-dessus
  • Descendant : Tous les éléments en-dessous

Héritage CSS

Propriétés Héritées

.parent {
font-family: Arial;
color: blue;
line-height: 1.5;
}
Parent avec styles
Enfant hérite des propriétés
Petit-enfant hérite aussi

font-family, color, line-height sont hérités

Propriétés Non Héritées

.parent {
background: lightblue;
border: 2px solid red;
margin: 20px;
}
Parent avec styles
Enfant n'hérite pas
Petit-enfant non plus

background, border, margin ne sont pas hérités

Forcer l'Héritage

.enfant {
background: inherit;
border: inherit;
margin: inherit;
}
Parent avec styles
Enfant force l'héritage
Petit-enfant aussi

Utilisez inherit pour forcer l'héritage

Non Héritées

  • background

  • margin
  • padding

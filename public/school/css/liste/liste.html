Reset CSS pour Listes

Reset Complet

ul, ol {
list-style: none;
margin: 0;
padding: 0;
}

li {
margin: 0;
padding: 0;
}

Supprime tous les styles par défaut

Reset Partiel

ul, ol {
margin: 0 0 1rem 0;
padding-left: 2rem;
}

li {
margin-bottom: 0.5rem;
}

Conserve une mise en page cohérente

Normalisation

ul, ol {
margin: 1rem 0;
padding-left: 2.5rem;
}

li {
line-height: 1.6;
}

Styles cohérents cross-browser

Bonnes Pratiques

  • Utilisez list-style: none pour les navigations
  • Préférez les marges aux paddings pour l'espacement
  • Utilisez des unités relatives (rem/em) pour le responsive
  • Testez l'accessibilité avec les lecteurs d'écran
  • Maintenez la sémantique HTML appropriée

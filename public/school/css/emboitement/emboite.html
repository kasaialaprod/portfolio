Le Box Model CSS

Comprendre la structure des boîtes en CSS

Structure du Box Model

Margin
Border
Padding
Content

De l'extérieur vers l'intérieur : Margin → Border → Padding → Content

Propriétés du Box Model

.element {
width: 300px;
height: 200px;
padding: 20px;
border: 5px solid;
margin: 30px;
}

Largeur totale :
width + padding-left + padding-right + border-left + border-right

Hauteur totale :
height + padding-top + padding-bottom + border-top + border-bottom

Types de Box Model

Content-Box (Défaut)

box-sizing: content-box;
Width: 200px
+ Padding: 20px
+ Border: 5px
= 250px total

La width/heigth ne incluent que le contenu

Border-Box

box-sizing: border-box;
Width: 200px
Inclut padding et border
= 200px total

La width/heigth incluent padding et border

Box-Sizing

Content-Box vs Border-Box

Content-Box (Défaut)

.element {
width: 200px;
padding: 20px;
border: 5px solid;
/* Largeur réelle : 250px */
}
200px
+20px padding
+5px border
= 250px

Border-Box

.element {
box-sizing: border-box;
width: 200px;
padding: 20px;
border: 5px solid;
/* Largeur réelle : 200px */
}
200px total
contenu: 150px
padding: 20px
border: 5px

Pratique Recommandée

Reset Global

* {
box-sizing: border-box;
}

Applique border-box à tous les éléments

Héritage

html {
box-sizing: border-box;
}

*, *::before, *::after {
box-sizing: inherit;
}

Permet de changer facilement pour des composants spécifiques

Effondrement des Marges

Margin Collapse Vertical

.box1 { margin-bottom: 30px; }
.box2 { margin-top: 20px; }
/* Espace total : 30px (pas 50px) */
margin-bottom: 30px
margin-top: 20px
Espace réel : 30px

Les marges verticales adjacentes se combinent

Éviter l'Effondrement

/* Solutions */
padding: 1px 0;
border: 1px solid transparent;
display: inline-block;
overflow: auto;
Avec padding
Avec border
Inline-block

Margin Collapse Parent-Enfant

.parent { margin-top: 20px; }
.child { margin-top: 30px; }
/* Les deux marges se combinent */
Parent (margin-top: 20px)
Enfant (margin-top: 30px)
Marge combinée : 30px

Cas sans Effondrement

/* Pas d'effondrement */
flexbox, grid
position: absolute/fixed
float

Flex item 1
Flex item 2

Règles d'Effondrement

  • Se produit seulement avec les marges verticales
  • Entre éléments frères adjacents
  • Entre parent et premier/dernier enfant
  • La plus grande marge l'emporte
  • Les marges négatives se soustraient

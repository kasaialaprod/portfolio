Contrôle du programme, boucles et tableaux

1. Entrées utilisateur : lire les données

  • On utilise scanf pour demander une valeur à l’utilisateur (nombre, char, texte…)
  • On donne toujours l’adresse (avec &) de la variable à remplir (sauf chaîne !)
int age;
printf("Quel âge as-tu ? ");
scanf("%d", &age);
printf("Tu as %d ans !\n", age);
Quel âge as-tu ? 18
Tu as 18 ans !

2. Structure conditionnelle (if, else)

  • On teste une condition, on définit des alternatives
if (age < 18) {
  printf("Mineur\n");
} else if (age < 60) {
  printf("Adulte\n");
} else {
  printf("Sénior\n");
}

3. switch (menu, interrogation sur des cas précis)

  • On définit plusieurs cas précis pour une variable
int choix = 2;
switch(choix) {
  case 1: printf("Attaquer\n"); break;
  case 2: printf("Défendre\n"); break;
  case 3: printf("Inventaire\n"); break;
  default: printf("Action inconnue\n");
}

4. Conditions multiples

  • && (ET), || (OU), ! (NON)
// Si mineur et joueur, accès spécial

if (age < 18 && choix == 1) {
  printf("-75% de réduction\n");
} else if (age < 18 || choix == 2) {
  printf("-50% de réduction\n");
} else {
  printf("Pas de réduction\n");
}

5. Boucles : while et for

  • while : répète tant que la condition est vraie
  • for : plus compact, pratique pour compter
// Compter de 0 à 4 :

for (int i=0; i<5; i++) {
  printf("%d ", i);
}

// Tant que x n'est pas zéro

int x = 5;
while (x > 0) {
  printf("%d ", x);
  x--;
}
0 1 2 3 4
5 4 3 2 1

6. Tableaux (“arrays”) & bouclage

  • Définir : int notes[3] = {12, 15, 8};
  • Les indices commencent à zéro (notes[0] = 12)
  • Boucle : on accède à chaque case via un for
int notes[3] = {12, 15, 8};
for (int i=0; i<3; i++) {
  printf("Note %d : %d\n", i+1, notes[i]);
}
Note 1 : 12
Note 2 : 15
Note 3 : 8

7. Chaînes de caractères & entrée utilisateur

  • Déclarer une chaîne : char nom[30];
  • scanf("%s", nom); lit un mot (attention, s’arrête à l’espace !)
  • Pour lire une ligne entière : fgets(nom, 30, stdin);(la valeur saisie sur la console est stockée dans la variable nom, y compris les espaces, et ce jusqu’à 29 caractères)
char nom[30];
printf("Quel est ton pseudo ? ");
scanf("%s", nom);
printf("Bienvenue, %s !\n", nom);
Quel est ton pseudo ? kasai2000
Bienvenue, kasai2000 !
// Variante : lire toute une ligne (avec espace)

char nom[30];
printf("Quel est ton pseudo ? ");
fgets(nom, 30, stdin);
printf("Bienvenue, %s !\n", nom);
Quel est ton pseudo ? Kasai le Dictateur
Bienvenue, Kasai le Dictateur !
